・「実行ファイル/Bounceback Blitz.exe」を開いて起動します。
・exeファイルと起動に必要なリソースファイルが入っています。


◆作品について◆
◇作品名        : Bounceback Blitz
◇ジャンル      : 横スクロール3Dアクション
◇使用言語      : C++
◇SDK           : DirectX9
◇開発期間      : 2023/10/02〜2023/10/27
◇GitHubURL     : https://github.com/SeigaSouma/2024_01Action.git
◇プレイ動画URL : https://youtu.be/Ouwjfsr3d30


◆作品概要◆
・『風のクロノア』のマップ設計×『LIBLADE』の斬撃システムを合体させたゲームです
・ツインスティックのアクションゲーム
・スティックを弾いた方向に斬撃が発生し、敵の弾を跳ね返すことが出来る
　跳ね返す方向を場に合わせて攻略しよう


◆実装内容◆
・自作モデルとの判定を取り、モデルに沿った起伏の判定
・マップに沿う移動実装の際、Catmull-Rom Spline補間を使用
・カメラマップに沿ったカメラワーク


◆操作方法◆ ※ツインスティック対応のコントローラーが必要になります
・左スティック：移動
・右スティック：攻撃
・LB：ジャンプ
・START：ポーズ
